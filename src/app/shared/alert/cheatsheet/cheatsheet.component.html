Hashcat uses UTF-8. If you are doing an attack using a wordlist, as soon as the format is supported, it should be fine. If you are trying a brute attack, it'll require using the command --hex-charset. Example:

German: --hex-charset -1 c3 -2 8489969c9fa4a9b6bc german.hash ?a?a?a?1?2?a?a

Encoding: It is better practice to encode the custom charsets in the mask. Please note From and To in the example, i.e., --encoding-from=utf-8 or --encoding-to=utf-8
